## Use this module when

- The request is about how a scene should look before code is written.
- You need visual critique or refinement without changing the runtime contract.

## Direct rules

- Load `references/authoring/visual/plan` for one-scene visual planning.
- Load `references/authoring/visual/critique` for visual review and diagnosis.
- Load `references/authoring/visual/refinement` for targeted visual fixes.
- Load `references/authoring/visual/common-errors` when a design is failing in predictable ways and needs fast correction.
